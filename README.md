# 🕒 Ultra-Minimalist Digital Clock

A sleek, full-screen digital clock designed for high visibility and a modern aesthetic. This project features a responsive design that looks great on both desktop monitors and mobile devices.

## ✨ Features
* **Massive Display:** Uses "Viewport Units" to ensure the clock fills the screen perfectly.
* **High Contrast:** Neon-cyan text on a deep black background for maximum readability.
* **Real-time Updates:** Synchronized every second using JavaScript.
* **Localized Date:** Automatically displays the current day, date, and year in Dutch (NL) formatting.
* **Responsive:** Automatically adjusts font sizes for smartphones and tablets.
